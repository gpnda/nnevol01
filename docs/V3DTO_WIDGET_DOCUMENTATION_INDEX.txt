🎯 ГЛАВНАЯ ИДЕЯ v3dto (30 секунд)
================================================================================

1. Виджеты получают ДО ЧЕРЕЗ RenderStateDTO (НИКОГДА не импортируй world!)
2. Инициализируй БЕЕЗ параметров (или только callback)
3. Всё определи как class constants (WIDGET_X, WIDTH, COLORS и т.д.)

Структура:
  class MyWidget:
      WIDGET_X = 10              # Constants
      WIDTH = 200
      COLORS = {...}
      
      def __init__(self):                            # БЕЗ параметров
          self.rect = pygame.Rect(...)
          self.surface = pygame.Surface(...)
      
      def draw(self, screen, render_state):          # Только эти параметры
          self.surface.fill(self.COLORS['background'])
          # рисуй на self.surface
          screen.blit(self.surface, (self.rect.x, self.rect.y))
