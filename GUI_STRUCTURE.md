"""
GUI Module Documentation

После рефакторинга модуль разделен на три класса:

1. VariablesPanel - Панель переменных
   ==========================================
   Отвечает за отображение и управление переменными приложения.
   
   Управление позицией и размерами (единообразно):
   - PANEL_X = 0              # Абсолютная X координата (левая граница)
   - PANEL_Y = 0              # Абсолютная Y координата (верхняя граница)
   - PANEL_WIDTH = 0          # 0 = весь экран, >0 = фиксированная ширина
   - PANEL_HEIGHT = 0         # 0 = весь экран минус нижняя панель, >0 = фиксированная высота
   - FUNC_KEYS_PANEL_HEIGHT = 80  # Высота нижней панели функциональных клавиш
   
   Смещения элементов внутри панели:
   - PADDING_X = 10           # Отступ слева и справа
   - PADDING_Y = 10           # Отступ сверху и снизу
   - TITLE_Y_OFFSET = 10      # Y смещение заголовка от верхнего края панели
   - TITLE_BOTTOM_OFFSET = 50 # Смещение первого элемента от верхнего края панели
   - ITEM_VALUE_X = 50        # X смещение значения переменной
   - HIGHLIGHT_*_OFFSET       # Смещения для выделения выбранного элемента
   
   Цвета:
   - 'panel_bg': (0, 0, 85)         # Фон панели
   - 'text': (170, 170, 170)        # Обычный текст
   - 'highlight': (255, 255, 255)   # Выделенный текст
   - 'selected': (0, 85, 0)         # Выбранный элемент
   - 'active': (0, 170, 0)          # Активный элемент (редактирование)


2. FunctionKeysPanel - Панель функциональных клавиш
   ==================================================
   Отвечает за отображение и управление функциональными клавишами.
   
   Управление позицией и размерами (единообразно):
   - PANEL_X = 0              # Абсолютная X координата
   - PANEL_Y = 0              # 0 = автоматически (экран_высота - PANEL_HEIGHT)
   - PANEL_WIDTH = 0          # 0 = весь экран, >0 = фиксированная ширина
   - PANEL_HEIGHT = 80        # Высота панели функциональных клавиш
   
   Смещения элементов внутри панели:
   - PADDING_X = 20           # Отступ элементов слева
   - PADDING_Y = 10           # Отступ элементов сверху
   - ROW_HEIGHT = 30          # Высота строки с функциональными клавишами
   - SEPARATOR_HEIGHT = 20    # Высота линии разделения выше панели
   - SEPARATOR_WIDTH = 2      # Толщина линии разделения
   
   Цвета:
   - 'text': (170, 170, 170)        # Обычный текст
   - 'func_keys': (170, 170, 0)     # Функциональные клавиши
   - 'border': (85, 85, 85)         # Линия разделения


3. BIOSStyleGUI - Главный класс GUI
   =================================
   Комбинирует обе панели в единый интерфейс.
   Отвечает за обработку событий и навигацию между панелями.
   
   Использует:
   - VariablesPanel для управления переменными
   - FunctionKeysPanel для управления функциональными клавишами


ЕДИНООБРАЗНЫЙ ПОДХОД К ПОЗИЦИОНИРОВАНИЮ
==========================================

Обе панели используют одинаковый принцип для управления положением и размерами:

1. Абсолютное позиционирование через PANEL_X и PANEL_Y
2. Гибкие размеры: 0 = авто, >0 = фиксированное значение
3. Все смещения элементов внутри панели явно определены
4. Легко изменяемо и понятно


ИСПОЛЬЗОВАНИЕ
=============

from gui import BIOSStyleGUI

# Создание GUI
gui = BIOSStyleGUI(screen, font_size=24, line_height=30)

# Добавление переменных
gui.add_variable("Speed", 50, min_val=0, max_val=100)
gui.add_variable("Power", 75.5, float, 0.0, 100.0)

# Добавление функциональных клавиш
gui.add_function_key("F1", "Save", save_callback)
gui.add_function_key("F2", "Load", load_callback)

# Работа с переменными
value = gui.get_variable("Speed")
gui.set_variable("Speed", 75)

# Отрисовка
gui.draw()

# Обработка событий
gui.handle_event(event)


ПРИМЕРЫ КАСТОМИЗАЦИИ
====================

Изменить позицию панели переменных на начало (слева):
    VariablesPanel.PANEL_X = 0
    VariablesPanel.PANEL_Y = 0

Сделать панель переменных фиксированной ширины (например, 600px):
    VariablesPanel.PANEL_WIDTH = 600

Изменить высоту панели функциональных клавиш:
    FunctionKeysPanel.PANEL_HEIGHT = 100
    VariablesPanel.FUNC_KEYS_PANEL_HEIGHT = 100

Увеличить отступы в панели переменных:
    VariablesPanel.PADDING_X = 20
    VariablesPanel.PADDING_Y = 20


ТЕСТИРОВАНИЕ
============

Для запуска тестов:
  python test_gui_split.py

Все тесты должны пройти успешно.
"""
